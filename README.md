# WeiCommunity
sina weibo third party client on android os
## 简介
新浪微博第三方安卓客户端，采用MVP模式开发，计划在还原微博UI的同时加入一些新特性。
## 目前进度
完成登陆功能、查看微博列表、切换Group、转发微博、评论微博、查看微博详细信息等。
